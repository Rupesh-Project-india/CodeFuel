Introduction to Go

Go, also known as Golang, is an open-source programming language developed by Google. It is statically typed, compiled, and designed for simplicity, concurrency, and performance.

Basic Syntax

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

Variables and Data Types

var name string = "Rupesh"
age := 20            // shorthand declaration
isStudent := true

// Data types: string, int, float64, bool, array, slice, map
numbers := []int{1,2,3,4,5}
person := map[string]string{"name": "Rupesh", "course": "Go"}

Conditional Statements

age := 20

if age >= 18 {
    fmt.Println("You are an adult")
} else {
    fmt.Println("You are a minor")
}

Loops

// For loop
for i := 1; i <= 5; i++ {
    fmt.Println("Number", i)
}

// While-like loop
count := 1
for count <= 5 {
    fmt.Println("Count", count)
    count++
}

// Range loop
numbers := []int{1,2,3,4,5}
for _, num := range numbers {
    fmt.Println(num)
}

Functions

func greet(name string) {
    fmt.Println("Hello,", name)
}

func main() {
    greet("Rupesh")
}

Conclusion

Go is efficient, simple, and great for building scalable applications. It is widely used for backend development, cloud services, and networking programs.
