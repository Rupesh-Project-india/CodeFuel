Introduction to Rust

Rust is a systems programming language focused on safety, speed, and concurrency. It prevents memory errors at compile-time and is ideal for performance-critical applications.

Basic Syntax

fn main() {
    println!("Hello, World!");

    let name: &str = "Rupesh";
    let age: u32 = 20;
}

Variables and Mutability

let x = 5;       // immutable
let mut y = 10;  // mutable
y += 5;

Conditional Statements

let age = 20;

if age >= 18 {
    println!("You are an adult");
} else {
    println!("You are a minor");
}

Loops

// For loop
for i in 1..=5 {
    println!("Number {}", i);
}

// While loop
let mut count = 1;
while count <= 5 {
    println!("Count {}", count);
    count += 1;
}

Functions

fn greet(name: &str) {
    println!("Hello, {}!", name);
}

fn main() {
    greet("Rupesh");
}

Conclusion

Rust is safe, fast, and excellent for building reliable systems and applications. Learning Rust helps you write memory-safe and high-performance programs.
