Introduction to Perl

Perl is a high-level, general-purpose programming language known for text processing and scripting. It is widely used for web development, system administration, and data manipulation.

Basic Syntax

#!/usr/bin/perl
use strict;
use warnings;

print "Hello, World!\n";

my $name = "Rupesh";
my $age = 20;

Conditional Statements

my $age = 20;

if($age >= 18) {
    print "You are an adult\n";
} else {
    print "You are a minor\n";
}

Loops

# For loop
for(my $i = 1; $i <= 5; $i++) {
    print "Number $i\n";
}

# While loop
my $count = 1;
while($count <= 5) {
    print "Count $count\n";
    $count++;
}

Subroutines (Functions)

sub greet {
    my ($name) = @_;
    print "Hello, $name!\n";
}

greet("Rupesh");

Conclusion

Perl is versatile, excellent for scripting and text manipulation, and widely used in legacy systems. Learning Perl allows you to handle files, strings, and data efficiently.
