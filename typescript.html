Introduction to TypeScript

TypeScript is a superset of JavaScript that adds static typing and modern features. It helps catch errors during development and makes code more maintainable.

Basic Syntax

// Display text
console.log("Hello, World!");

// Variables with types
let name: string = "Rupesh";
let age: number = 20;
const isStudent: boolean = true;

// Functions
function greet(person: string): void {
    console.log("Hello, " + person);
}

greet(name);

Conditional Statements

let age: number = 20;

if(age >= 18){
    console.log("You are an adult");
} else {
    console.log("You are a minor");
}

Loops

// For loop
for(let i: number = 1; i <= 5; i++){
    console.log("Number " + i);
}

// While loop
let count: number = 1;
while(count <= 5){
    console.log("Count " + count);
    count++;
}

Interfaces

interface Person {
    name: string;
    age: number;
}

const rupesh: Person = { name: "Rupesh", age: 20 };
console.log(rupesh.name);

Conclusion

TypeScript enhances JavaScript with type safety and better tooling. It is widely used in large-scale applications and frameworks like Angular.
