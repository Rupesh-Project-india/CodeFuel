Introduction to PHP

PHP (Hypertext Preprocessor) is a server-side scripting language designed for web development. It is used to create dynamic web pages and interact with databases.

Basic Syntax

<?php
// Display text
echo "Hello, World!";

// Variables
$name = "Rupesh";
$age = 20;

// Comments
// This is a single-line comment
/* This is a multi-line comment */
?>

    echo "You are an adult";

Loops

// For loop
for($i = 1; $i <= 5; $i++){
    echo "Number ".$i."<br>";
}

// While loop
$count = 1;
while($count <= 5){
    echo "Count ".$count."<br>";
    $count++;
}

// Foreach loop (arrays)
$numbers = array(1,2,3,4,5);
foreach($numbers as $num){
    echo $num."<br>";
}

Functions

function greet($name){
    echo "Hello, ".$name."!";
}

greet("Rupesh");

Conclusion

PHP is widely used for server-side web development. Learning PHP allows you to create dynamic websites and interact with databases efficiently.
