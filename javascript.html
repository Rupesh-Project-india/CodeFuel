Introduction to JavaScript

JavaScript is a high-level, interpreted programming language used to make web pages interactive. It works alongside HTML and CSS and is essential for web development.

Basic Syntax

// Display text
console.log("Hello, World!");

// Variables
let name = "Rupesh";
const pi = 3.14;
var age = 20;

// Comments
// This is a single-line comment
/* This is a multi-line comment */

    console.log("You are an adult");

Loops

// For loop
for(let i = 1; i <= 5; i++){
    console.log("Number " + i);
}

// While loop
let count = 1;
while(count <= 5){
    console.log("Count " + count);
    count++;
}

// ForEach loop (arrays)
[1,2,3,4,5].forEach(num => console.log(num));

Functions

function greet(name){
    console.log("Hello, " + name + "!");
}

greet("Rupesh");

Conclusion

JavaScript is essential for creating interactive and dynamic web pages. Learning JavaScript allows you to build modern web applications, games, and more.
