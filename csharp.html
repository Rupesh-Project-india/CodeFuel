Introduction to C#

C# (C-Sharp) is a modern, object-oriented programming language developed by Microsoft. It is widely used for building desktop applications, web applications, games (with Unity), and enterprise solutions.

Basic Syntax

using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, World in C#!");
    }
}

Features of C#

  • Simple and Modern Language
  • Object-Oriented Programming
  • Rich Standard Library
  • Type-Safe and Secure
  • Cross-Platform (with .NET Core)
  • Used in Game Development (Unity Engine)

Example: Variables and Data Types

using System;

class Program {
    static void Main() {
        int age = 21;
        string name = "Rupesh";
        double pi = 3.14159;
        bool isCoder = true;

        Console.WriteLine("Name: " + name);
        Console.WriteLine("Age: " + age);
        Console.WriteLine("Value of Pi: " + pi);
        Console.WriteLine("Coder? " + isCoder);
    }
}
