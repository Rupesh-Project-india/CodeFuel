Introduction to Python

Python is a high-level, interpreted programming language known for its readability and simplicity. It is widely used in web development, data science, automation, and AI.

Basic Syntax

# Print text
print("Hello, World!")

# Comments
# This is a single-line comment
""" This is
a multi-line comment """

Variables and Data Types

# Variables
name = "Rupesh"
age = 20
price = 19.99
is_student = True

# Data types: str, int, float, bool, list, dict, tuple
numbers = [1, 2, 3, 4, 5]
person = {"name": "Rupesh", "age": 20}

Conditional Statements

age = 20

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

Loops

# For loop
for i in range(1, 6):
    print("Number", i)

# While loop
count = 1
while count <= 5:
    print("Count", count)
    count += 1

Functions

def greet(name):
    print("Hello,", name)

greet("Rupesh")

Conclusion

Python is beginner-friendly and versatile. Learning Python opens doors to web development, data analysis, automation, and more.
