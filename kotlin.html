Introduction to Kotlin

Kotlin is a modern, statically typed language used for Android development and JVM applications.

Basic Syntax

fun main() {
    println("Hello, World!")
    
    val name: String = "Rupesh"
    var age: Int = 20
}

Conditional Statements

val age = 20

if (age >= 18) {
    println("You are an adult")
} else {
    println("You are a minor")
}

Loops

// For loop
for (i in 1..5) {
    println("Number $i")
}

// While loop
var count = 1
while(count <= 5) {
    println("Count $count")
    count++
}

Functions

fun greet(name: String) {
    println("Hello, $name!")
}

greet("Rupesh")

Conclusion

Kotlin is concise, safe, and fully interoperable with Java.
