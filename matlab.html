MATLAB

MATLAB (Matrix Laboratory) is a high-level programming language and interactive environment primarily used for numerical computing, data analysis, simulations, and visualization. It is widely used in engineering, mathematics, scientific research, and AI applications.

Key Features of MATLAB

  • Powerful mathematical and matrix operations
  • Built-in visualization and graphing tools
  • Toolboxes for Machine Learning, Image Processing, Control Systems, etc.
  • Easy integration with C, C++, Java, and Python

Basic Syntax

% MATLAB is case-sensitive
a = 10;         % Assign a value
b = 20;
sum = a + b     % Addition
disp(sum)       % Display output

Example: Plotting a Graph

x = 0:0.1:10;        % Define range
y = sin(x);          % Calculate sine values
plot(x, y);          % Plot graph
title('Sine Wave');  % Add title
xlabel('X-axis');    % Label axes
ylabel('Y-axis');
grid on;             % Add grid
