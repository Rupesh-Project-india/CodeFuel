Introduction to Bash

Bash (Bourne Again SHell) is a command-line shell used in Linux and Unix systems. It allows you to execute commands, automate tasks, and write shell scripts.

Basic Commands

# List files in a directory
ls

# Print current directory
pwd

# Display text
echo "Hello World"

# Create a directory
mkdir myfolder

# Remove a file
rm file.txt

Variables in Bash

# Assign a variable
name="Rupesh"

# Access variable
echo $name

# Read input from user
read -p "Enter your name: " user_name
echo "Hello $user_name"

Conditional Statements

if [ $age -ge 18 ]; then
  echo "You are an adult"
else
  echo "You are a minor"
fi

Loops in Bash

# For loop
for i in 1 2 3 4 5
do
  echo "Number $i"
done

# While loop
count=1
while [ $count -le 5 ]
do
  echo "Count $count"
  count=$((count + 1))
done

Writing a Bash Script

#!/bin/bash
# My first script
echo "Welcome to Bash scripting!"

Conclusion

Bash is a powerful tool for system administration and automation. Learning Bash helps you work efficiently in Linux/Unix environments.
